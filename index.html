Stage 0 Concepts

Web Interworkings
HTML is the basis for every webpage. Hyperlinks are links between webpages. Files that can be found on the web include: Plain Text, HTML, Images, Videos, and Music.

Major components of the web include browsers, servers, the internet and HTTP. Browsers read and interpret HTML to display what we see. Servers host webpage content and the internet is a compilation of connected routers that are traversed to communicate with desired servers.

HTML
HTML stands for Hypertext Markup Language. It is the language of the web, There are several components to HTML: text content (what we see), markup (what it looks like), and references to other documents (images, video). This video gives a good overview once the narrator finishes explaining his qualifications.

Tags and Elements of HTML
Elements make up an HTML document. In HTML, tags are used to tell browsers what type of content they are reading. HTML elements can be either inline or block. Block elements are contained in an invisible box.

In general, it is good to keep in mind that computers are very literal. This characteristic can turn a small typo into hours of debugging.
